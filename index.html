Git Is A Version Control System

Starting With Basics

git init command

it is used to get all the files on the backstage but not tracking them

git add command

Now The Files Are Tracked And Ready To Go Into The Stage

git commit -m "add your message"

Now The Git Will Be Pushing Your Changes In To The Version

git log

Gives All The Changes With Some Code To Go To Previous One

git reset --hard "code"

The Code Can Be Shown ON The git log command

You Can Go Back To Previous Code With Deleting All The Code OF the Previous Commit

git reset --soft "code"

The Code Can Be Shown ON The git log command

You Can Go Back To Previous Code With not deleting All The Code but by deleting the commit OF the Previous Commit
